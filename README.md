# Hritik | AI/ML Engineer Portfolio

A futuristic, AI-driven portfolio website designed to showcase projects, skills, and academic achievements in the fields of Deep Learning, Computer Vision, and Natural Language Processing.

## 🚀 Features

* **Dark Futuristic UI:** Deep navy and charcoal black background with electric blue and neon cyan glowing accents.
* **Interactive Elements:**
    * Neural network particle background with mouse attraction.
    * Interactive hover states on skill cards, domain boxes, and project buttons.
    * Smooth scroll reveal animations across all sections.
* **Comprehensive Sections:**
    * **Hero:** Dynamic introduction with floating tech badges.
    * **About:** Professional summary with a mini model accuracy chart.
    * **Skills:** Animated skill progress bars and domain expertise breakdown.
    * **Projects:** Detailed project cards featuring visual previews (e.g., Confusion Matrix, Accuracy Ring).
    * **Research:** Highlighting certifications from Oracle and research papers.
    * **Experience:** Vertical, color-coded timeline tracking internships, education, and key milestones.
    * **Contact:** Functional contact form layout with social links.
* **Fully Responsive:** Optimized for desktops, tablets, and mobile devices with a slide-in hamburger menu.

## 🛠️ Built With

* **HTML5:** Semantic structure and layout.
* **Vanilla CSS (CSS3):** Custom variables, Flexbox/Grid layouts, gradient text, and complex keyframe animations (no external CSS framework).
* **Vanilla JavaScript (ES6+):** Handling particle canvas, Intersection Observers for scroll animations, count-up stats, and mobile navigation.
* **[Lucide Icons](https://lucide.dev/):** For clean, modern SVG iconography.

## 💻 Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/HritikGodara/[repository-name].git
   ```
2. Navigate to the project directory:
   ```bash
   cd [repository-name]
   ```
3. Run a local server to view the site. For example, using Python:
   ```bash
   python -m http.server 8080
   ```
4. Open your browser and visit `http://localhost:8080`.
